# Vocabulary in Use — Advanced · жаттығу қосымшасы

Оффлайн, серверсіз, build-сыз жаттығу құралы. Экранның бір жағында PDF, екінші жағында осы қосымша.

## Қалай ашу

**`index.html` файлын браузерде екі рет бас — бітті.** Дерек оның ішіне ендірілген, сондықтан
ештеңе сұрамайды, интернет те, сервер де керек емес.

## Файлдар

| Файл | Не үшін |
|---|---|
| `index.html` | **Осыны ашасың.** Дерегі ішіне ендірілген дайын қосымша (бір файл). |
| `app.template.html` | Бастапқы код (CSS + JS). Мұнда дерек жоқ — өңдеу осында жасалады. |
| `exercises.json` | 101 бөлім · 458 жаттығу · 2338 сұрақ. |
| `build.py` | `app.template.html` + `exercises.json` → `index.html`. |

Кодты немесе JSON-ды өзгертсең, `index.html`-ді қайта жина:

```bash
python3 build.py
```

> `index.html`-ді тікелей түзетудің қажеті жоқ — ол әр жинауда қайта жазылады.

## Мүмкіндіктер

- **Бөлімдер тізімі** — прогресс жолағы, іздеу (нөмір немесе атау бойынша), PDF бет нөмірлері (жаттығу / сөздік).
- **Тексеру** — `Enter` немесе `✓`. Регистр, тыныс белгі, сызықша, апостроф, `…` еленбейді. `/` арқылы берілген барлық нұсқалар және жақшадағы бөліктің бар/жоқ нұсқалары қабылданады.
  - Дұрысқа өте жақын жауап (әріп қатесі) → сары «өзің шеш» күйі, `Дұрыс деп есепте` / `Қате деп есепте` батырмалары.
  - `exampleAnswers: true` жаттығуларда дәл сәйкессіздік қате саналмайды — кітаптың үлгісі көрсетіліп, шешім өзіңе қалады.
  - Қате болса — толық дұрыс жауап көрсетіледі, `Шын мәнінде дұрыс жаздым` батырмасы бар.
- **👁** — жауапты көрсету (ұпайға есептелмейді), **↻** — қайта әрекет.
- **Мастер логикасы** — қатарынан **3 рет** дұрыс жауап → сөйлем меңгерілді (✓). Қате болса, санақ нөлге түседі және сөйлем «Қателерім» тізіміне қайта түседі.
- **Қателерім** — меңгерілмеген барлық сөйлемдер: бөлімдер бойынша немесе «Кездейсоқ 20» аралас режимде.
- **Статистика** — жалпы прогресс, дұрыс жауап үлесі, соңғы 14 күн диаграммасы, бөлімдер кестесі, прогресті `.json`-ға сақтау / қалпына келтіру / өшіру.
- **⚠️ PDF-пен салыстыр** белгісі — сол жаттығудың мәтіні PDF-тен көшірілгенде бет-беттілік күрделі болған (кесте, диалог, көп баған). Белгінің үстіне тінтуірді апарсаң, нақты ескерту мәтіні шығады.
- **Ашық сұрақ (Over to you)** жаттығулары ұпай санамайды, тек ескерту көрсетеді.
- Сөздер қорабындағы сөзді бассаң — ол ағымдағы өріске қосылады.

## Прогресс қайда сақталады

Браузердің `localStorage` жадында (`vocabAdv.progress.v1`). Браузерді жапсаң да жоғалмайды, бірақ:
- басқа браузерде / профильде — бөлек;
- `file://` мен `http://localhost` — бөлек жады;
- `index.html` файлын басқа қалтаға жылжытсаң — жаңа жол, жаңа жады (прогресс «жоғалғандай» көрінеді);
- браузер деректерін тазаласаң — өшеді. Сақтық үшін Статистика бетінен `.json` көшірмесін жүктеп ал.
