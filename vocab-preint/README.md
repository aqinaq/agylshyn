# English Vocabulary in Use — Pre-Intermediate (офлайн жаттығу)

Толығымен офлайн жұмыс істейтін лексика жаттығу қосымшасы. Сервер де, build те,
интернет те қажет емес — `index.html` файлын браузерде қос шертіп ашсаң болғаны.

## Іске қосу

```
open index.html          # macOS
```

Экранның оң жағында PDF-ті, сол жағында осы бетті аш. Суреттер мен кестелер PDF-те
қалады — әр unit-тің PDF беті чип түрінде көрсетіледі (📄 жаттығу беті,
📖 теория беті).

## Файлдар

| Файл | Не үшін |
|---|---|
| `index.html` | Беттің қаңқасы |
| `style.css` | Стильдер (ашық/қараңғы режим — жүйелік баптауға қарай автоматты) |
| `app.js` | Барлық логика: тексеру, прогресс, роутинг |
| `data.js` | `exercises.json`-ның `window.EVU_DATA = {...}` түріндегі нұсқасы |
| `exercises.json` | Дерек: 100 unit, кітап PDF-інен шығарылған |
| `tools/` | Деректі PDF-тен қайта құратын скрипттер + регрессия тесті |

`file://` арқылы ашқанда `fetch()` кейбір браузерде CORS себебімен істемейді,
сондықтан дерек `data.js` арқылы тікелей `<script>`-пен жүктеледі.

**`exercises.json` өзгерсе, `data.js`-ті қайта жасау керек:**

```
python3 -c "import json;d=json.load(open('exercises.json',encoding='utf-8'));open('data.js','w',encoding='utf-8').write('window.EVU_DATA = '+json.dumps(d,ensure_ascii=False,separators=(',',':'))+';')"
```

## Дерек қайдан алынған

Дерек кітаптың PDF-інен автоматты түрде шығарылған:

- **Сұрақтар** — жаттығу беттерінен;
- **Жауаптар** — кітаптың соңындағы Answer key бөлімінен (ол таза нөмірленген
  тізім болғандықтан, нөмірлеу дауласқанда Answer key басым саналады).

Кітап мәтінді **бағаналап** басады, ал кейбір жаттығуда item нөмірі оң жақ шетте
тұрады да, сөйлем сол жақта қалады (мыс. 4.2, 19.4). PDF-ті қарапайым оқығанда
нөмір мен сөйлем бір-біріне қате қиысады. `tools/extract_ex.py` мұны жолдарды
координатасы бойынша қайта құрастырып шешеді.

Қайта құру (PyMuPDF керек: `pip3 install pymupdf`):

```
python3 tools/extract_key.py     # Answer key -> tools/key.json
python3 tools/extract_ex.py      # жаттығу беттері -> tools/ex_raw.json
python3 tools/build.py           # екеуін біріктіру -> exercises.json
```

`tools/report.txt` — қолмен қарауды қажет ететін жерлер (қазір 5-еуі: суретке
қарап жауап беретін жаттығулар, оларда мәтін болмайды).

## Қалай жұмыс істейді

### Жаттығу түрлері

- **items** — негізгі, автоматты тексерілетін жаттығулар.
- **text** — нөмірленген бос орындары бар тұтас мәтін («Complete the text»).
  Мәтін жоғарыда бір рет көрсетіледі, астында нөмірленген өрістер.
- **open** — «Over to you». Жауап кілті жоқ, өзің ✓/✗ деп белгілейсің.
- **freeform** — санатқа бөлу / сөз тізімі. «Жауапты көрсету» батырмасы бар.
- **crossword** — тек ескерту, торды PDF-тен шешесің.

Сәйкестендіру жаттығуларында («Match the words…») оң жақтағы a/b/c нұсқалары
бөлек тізім болып бір рет көрсетіледі, сұраққа жабыспайды.

Item белгілері: `isExample` — кітаптың дайын мысалы (тексерілмейді, прогреске
кірмейді); `exampleAnswers` немесе `answer: null` — өзің тексересің.
«Over to you» жауаптары кітапта тек **үлгі** болғандықтан, олар ешқашан
автоматты тексерілмейді.

### Жауапты тексеру

1. **Нормалау** — кіші әріп, тырнақша/апостроф бір түрге келеді, сосын әріп пен
   саннан басқасының бәрі (бос орын, тыныс белгі) алынып тасталады.
   `"He's tying"` → `hestying`. Сондықтан үтір, нүкте, артық бос орын кедергі емес.
2. **Жақша** — `eat (any) lunch` десе, `eat lunch` те, `eat any lunch` те дұрыс.
3. **Балама жауаптар** — `/` және бөлек тұрған ` or ` арқылы бөлінеді:
   `comes out /is published` → екеуі де қабылданады. Сөз ішіндегі `/` те
   ескеріледі: `took/got a train` → `took a train`, `got a train`.

Ескерту: `;` арқылы бөлінген жауаптар — бір емес, **бірнеше бос орынның** жауабы
(мыс. `degrees; zero; extremely`), сондықтан олар бөлінбейді — бәрін теру керек.
Күрделі жағдайда қате шықса, **«Мен дұрыс жаздым»** батырмасы бар.

### Прогресс

- Қатарынан **3 рет** дұрыс жауап → item «★ меңгерілді» болады.
- Қате жауап → streak нөлге түседі, item «Қателер» бетіне түседі.
- Бәрі `localStorage`-те (`evu_v1` кілті) сақталады. Терілген мәтін тексерілмесе де
  сақталады.
- Прогресс пайызына `items` және `text` түрлері кіреді (жауап кілті барлары).

### Беттер

- **Units** — unit беті: барлық тапсырмалар, unit соңында ұпай мен «Барлығын тексеру».
- **Қателер** — қате жіберілген әрі әлі меңгерілмеген сұрақтар unit бойынша топталған.
- **Статистика** — жалпы карточкалар + әр unit бойынша кесте.

### Пернетақта

- **Enter** — жолды тексеріп, келесі өріске өтеді.

## Тест

`tools/test.html` — регрессия тесті (43 тексеру). Браузерде ашсаң, нәтиже беттің
астында шығады. Автоматты жүргізу:

```
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless \
  --allow-file-access-from-files --virtual-time-budget=40000 \
  --user-data-dir=/tmp/evu-test --dump-dom "file://$PWD/tools/test.html" \
  | sed -n '/===TESTS===/,/===END===/p'
```
