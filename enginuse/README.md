# Grammar Practice — English Grammar in Use

PDF-пен қатар қолдануға арналған жауап парағы. Сұрақ сөйлемдері де
көрсетіледі, бірақ суреттер мен кестелер PDF-те қалады — сондықтан
оң жақта PDF ашық тұрғаны ыңғайлы.

## Ашу
`index.html` файлын браузерде аш (қос шерту жеткілікті — сервер керек емес).

## Файлдар
- `index.html` — қаңқа
- `style.css` — дизайн (light/dark автоматты)
- `app.js` — логика (тексеру, прогресс, қателер, статистика)
- `data.js` — деректер (`exercises.json`-нан жасалған, `window.EGU_DATA`)
- `exercises.json` — бастапқы дерек

`exercises.json` өзгерсе, `data.js`-ті қайта жасау:

    python3 -c "import json;d=json.load(open('exercises.json',encoding='utf-8'));open('data.js','w',encoding='utf-8').write('window.EGU_DATA = '+json.dumps(d,ensure_ascii=False,separators=(',',':'))+';')"

## Дерек құрылымы
`exercises.json` — 145 unit, 566 тапсырма, 4756 item. Item өрістері:

- `question` — сұрақ сөйлемі (бос орынмен). PDF-тен алынған, кей жері шала.
- `answer` — кітап кілтіндегі толық жауап. `null` болса — өз ойыңмен жазатын item.
- `blank` — тек бос орынға жазылатын қысқа форма (мыс. `answer: "He’s tying"`,
  `blank: "’s tying"`). Екеуі де дұрыс саналады.
- `isExample: true` — кітаптағы дайын мысал: жауап өрісі жоқ, мәтіні бірден
  көрінеді, прогреске кірмейді (542 дана).

Тапсырманың өзінде `options` болуы мүмкін — сәйкестендіру жаттығуларының
a–h нұсқалары, олар сұрақтардың үстінде қорап болып шығады (14 тапсырма).

Жаттығу item саны: **4214**, оның 279-ында жауап кілті жоқ.

## Қолдану
- Сол жақтан unit таңда → 📄 PDF беті көрсетіледі, PDF-те сол бетті аш.
- Жауапты тер → **Enter** (тексереді әрі келесі жолға өтеді) немесе «Тексеру».
- Жауап кілті жоқ item-дер (өз ойыңмен жазатындар) — «✓ Дұрыс / ✗ Қате» өзің белгілейсің.
- Тексеру икемді: үлкен/кіші әріп, тыныс белгісі, апостроф, `/` және `or` арқылы
  берілген баламалар, `(жақшадағы)` міндетті емес бөліктер ескеріледі.
  Қателессе де «Мен дұрыс жаздым» деп қайта белгілеуге болады.
- Қате жіберілген item «Қателер» бетіне түседі, қатарынан **3 рет** дұрыс болса
  «меңгерілді» (★) болып тізімнен шығады.
- Прогресс localStorage-та (`egu_v1`) сақталады.
