# agylshyn

Cambridge-тің алты оқулығының жаттығулары — браузерде істейтін, серверсіз
жаттығу сайты. Сұрақтар PDF-тен шығарылып JSON-ға жиналған, жауап бірден
тексеріледі, прогресс `localStorage`-та сақталады.

## Кітаптар

| Кітап | Деңгей | Сұрақ |
|---|---|---|
| Essential Grammar in Use | Elementary | 2 408 |
| English Grammar in Use | Intermediate | 4 214 |
| Advanced Grammar in Use | Advanced | 2 465 |
| Vocabulary in Use — Pre-Intermediate | Pre-Int | 2 389 |
| Vocabulary in Use — Upper-Intermediate | Upper-Int | 2 626 |
| Vocabulary in Use — Advanced | Advanced | 2 327 |

Барлығы 16 005 сұрақ (essential-grammar-дің 424 «елес» жолы алынып,
vocab-upint нағыз Upper-Intermediate мазмұнына ауыстырылды — төмендегі
«Аудиттен кейінгі жөндеу» бөлімін қара).


## Іске қосу

`fetch()` қолданылатындықтан файлды қос шертіп ашуға болмайды — сервер керек:

```
cd site && python3 -m http.server 8777
# http://localhost:8777
```

Деректерді бастапқы қалталардан қайта жинау:

```
python3 site/tools/build_data.py
```

## Мүмкіндіктері

- Жауапты бірден тексеру, қателер беті, streak / mastered статистикасы
- Сөздік: сөзді меңзе (немесе қос шерт, ұзақ бас) — қазақша аудармасы шығады.
  1 385 сөздік ядро офлайн істейді, қалғаны API арқылы табылып кэштеледі
- Оқулық PDF-і сол беттің ішінде, керекті бетінен ашылады
- Интерфейс екі тілде: қазақша / ағылшынша
- Тақырыбы екеу: ашық / қараңғы
- Прогресс уақыт белгісімен жазылады: «бүгін», күнделікті серия (streak),
  соңғы 30 күн графигі, бөлім бойынша әлсіз тұстар, қайталау кестесі
- Дәлдік пен қамту бөлек көрсетіледі; прогресті JSON-ға сақтап/қайтаруға болады
- Офлайн істейді (PWA — Service Worker кэштейді)

Толық құжаттама — [site/README.md](site/README.md).

## Мазмұн сапасы

[AUDIT.md](AUDIT.md) — әр жауап оқулықтың өз «Key to Exercises» бөлімімен
салыстырылған (тексеру кезіндегі жағдай). Қысқаша: grammar, vocab-preint,
vocab-adv (8 930 сұрақ) — 99.5 %+ дәлдік; advanced-grammar — 93.8 %;
essential-grammar — сканерден алынған, сұрақтар PDF-тен оқылады.

## Аудиттен кейінгі жөндеу

Аудитте табылған мәселелер бойынша жасалғаны ([AUDIT.md](AUDIT.md) §8 реті):

- **vocab-upint** енді нағыз Upper-Intermediate мазмұны (101 юнит, 2 626 сұрақ,
  өз PDF кілтімен 100 %). Бұрын Pre-Intermediate көшірмесі болатын.
- **essential-grammar**-дің 424 «елес» жолы (сұрағы да, жауабы да жоқ) алынды —
  статистика енді шынайы (1 984 нақты сұрақ).
- **grammar**-дің юнит атаулары мен бөлімдері PDF-тің Contents бетінен қайта
  оқылды (36 қате атау түзелді); grammar мен vocab-upint-ке `section` қосылды.
- **Жауап тексеру:** үтірлі тізімдер енді ретсіз салыстырылады
  (729 сұрақ 0 % → 100 %); беттің «Key to Exercises» іздері жауаптан кесілді.
- **Прогресс сенімді:** жауап бет жабылғанда бірден жазылады
  (`visibilitychange`/`pagehide`), екі қойынды бір-бірін өшірмейді
  (`storage` оқиғасында біріктіру), әр жазбада уақыт белгісі (`ts`) бар.
- **Статистика:** Дәлдік / Қамту / Меңгерілді / Бүгін карточкалары, 30 күндік
  график, бөлім бойынша жинақ, сұрыпталатын кесте, «қайталауға дайын» тізім.
- **Қателер беті:** жауап жаңа әрекетке дейін жасырылады, ескі қате өріске
  толмайды («өткен жолы: …» болып бөлек), нұсқаулық әр жаттығуда бір рет.
- **Қолжетімділік:** кіріс өрістерінде `aria-label`, нәтижеде `aria-live`,
  кестеде пернетақта, белсенді таб/юнитте `aria-current`, модальда фокус қақпаны.
- **Мобиль:** табтар астыңғы жолаққа шықты — жоғарғы жолақ бір қатар, кітап
  атауы қиылмайды. Үш «Vocabulary in Use» деңгеймен ажыратылды.
- **Экспорт/импорт**, **офлайн (PWA)** қосылды.
- **Кейінге қалдырылды:** мәтіндегі OCR бос орындарын (`de grees`) автоматты
  жөндеу — сөздіксіз қауіпсіз істелмейді (дұрыс мәтінді бүлдіру қаупі);
  салыстыруға әсер етпейді. Авторлық құқық (Т6) — PDF-терді жариялау шешімі
  иесінде.
